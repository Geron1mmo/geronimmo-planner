=== Geronimmo Planner — Швидкий запуск ===

WINDOWS (найпростіше):
  Подвійний клік: D:\Geronimmo_Planner\RUN_GERONIMMO.bat
  АБО напряму: D:\Geronimmo_Planner\executables\windows\Geronimmo_Planner\geronimmo_planner.exe

ANDROID:
  Файл APK: D:\Geronimmo_Planner\executables\android\Geronimmo_Planner.apk
  Скопіюй на телефон і встанови.

Якщо треба перезібрати:
  cd D:\Geronimmo_Planner
  set JAVA_HOME=C:\Program Files\Android\Android Studio\jbr
  flutter build windows --release
  flutter build apk --release
